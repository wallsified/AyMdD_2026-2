  1. ¿Cuáles son las entidades con mayor incidencia de desapariciones?

Medidas de Heterogeneidad

def entropia_shannon(serie):
    p = serie.value_counts(normalize=True, dropna=True)
    return -np.sum(p * np.log(p))

def indice_gini(array):
    array = array.dropna().values
    if len(array) == 0:
        return np.nan
    array = np.sort(array)
    n = len(array)
    return (np.sum((2 * np.arange(1, n+1) - n - 1) * array)) / (n * np.sum(array))

Averiguaciones Relevantes (Continuación)

  • El fenómeno de las desapariciones está altamente concentrado en pocas entidades. Aunque el promedio nacional por estado es de ~2,700 casos, Tamaulipas (11,786) y Estado de México (10,935) lo superan ampliamente.
  • 2007 presenta el pico histórico de desapariciones con 10,410 casos, lo que estadísticamente es un outlier masivo frente a años previos. Este valor refleja un periodo de alta inseguridad en el país.
  • Aunque 2007 tiene el máximo absoluto de casos, la mediana temporal de desapariciones se ubica en 2016. Dado que los datos van de 1967 a 2023, esto implica que el 50 % de las desapariciones ocurrió solo en el periodo 2016–2023.
  • La columna estatus_busqueda tiene 27,364 valores nulos. Se sugiere que no son faltantes aleatorios (MCAR), sino MNAR, probablemente asociados a limitaciones de recursos o capacidad institucional para dar seguimiento.

Averiguaciones Relevantes (Continuación)

  • Destacan fuertes correlaciones positivas en el Noreste (Nuevo León–Tamaulipas, r = 0.87) y en el Pacífico (Colima–Nayarit, r = 0.87), indicando oleadas simultáneas de desapariciones.
  • La categoría “Con indicios / Ubicada” suma solo 12,563 registros en estatus_busqueda. En cambio, “Se busca reportante” (22,704) y “Se requieren datos de identidad” (18,362) dominan la distribución, evidenciando que la mayoría de los casos no avanzan por falta de información o abandono.
  • No se observan muchos homónimos pese al tamaño del dataset. La variabilidad de datos entre registros con el mismo nombre ayuda a distinguir personas únicas que podrían haber pasado desapercibidas.
  • Aunque el indicador de Tabasco es relevante, Estado de México y Sinaloa muestran la mayor correlación en el número de desaparecidos a lo largo del tiempo. Esto podría sugerir una conexión problemática entre ambas entidades, como operaciones ilegales compartidas.
